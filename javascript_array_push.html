JavaScript Array Push

To start with, we shall initialize an array. In this case, we define an array (called "colours") that has 4 attributes, all being named after fruits.

Definition: The push function is used to push a new element into the array such that the element is merged into the array, at the location subsequent to the last element. The length of the array is incremented, and the increased length is returned byt the push("") function.

NOTE: In the event that we need to push a given element onto tyhe front of an array, we need to use the unshift("") method
